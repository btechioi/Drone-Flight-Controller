## 🛸 Overview

A complete quadcopter flight controller for ESP32 featuring Madgwick/Mahony AHRS, attitude PID control, ESP-NOW based radio control, telemetry, and failsafe — all in a single Arduino sketch.

---

## 🛠️ Features

- Madgwick & Mahony AHRS sensor fusion
- Attitude PID controller (Roll, Pitch, Yaw)
- ESP-NOW wireless RC link
- Serial telemetry output
- Failsafe and safety checks
- MPU6050 IMU support
